# QTen Documentation

This site is generated with MkDocs and `mkdocstrings` directly from the
repository source tree.

The API reference is configured for NumPy-style docstrings so existing
`Parameters`, `Returns`, `Raises`, and `Examples` sections render without
switching the codebase to another documentation format.

Use this site for two things:

- User-facing guidance for installing and previewing docs locally.
- Generated API pages for both `qten` and `qten_plots`.

The generated reference lives under `API Reference` in the navigation sidebar.
